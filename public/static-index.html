OMNITRADE

OmniTrade Terminal

A modern trading terminal with a TabTrader-inspired interface, featuring draggable modules and a customizable workspace.

View on GitHub Documentation

Features

Modular Interface

Drag and drop components to create your ideal trading workspace. Arrange charts, order books, and watchlists exactly how you want them.

Mock Trading Data

This showcase version uses realistic mock data to demonstrate the UI and functionality without requiring a backend server.

TabTrader-Inspired

Inspired by TabTrader's intuitive layout, our terminal provides a familiar experience for cryptocurrency traders.

Theme Support

Switch between light and dark themes to suit your preference and reduce eye strain during long trading sessions.

Responsive Design

The terminal adapts to different screen sizes, providing an optimal experience on both desktop and mobile devices.

Open Source

The entire project is open source, allowing for community contributions and transparency in development.
